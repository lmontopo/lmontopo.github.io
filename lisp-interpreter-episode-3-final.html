Lisp Interpreter: Episode 3 (Final!)

Sun 16 November 2014

Let me start off by apologizing for the delay. I've been hesitant to start this blog post for several reasons, including:

  1. Presenting and explaining my entire lisp interpreter is a BIG (and therefore daunting) task.
  2. There are parts of my code that I'm not entirely satisfied with and I may refactor these parts in the future.
  3. The parts of my code that I'm unsatisfied with are pretty ugly and are a pain to read. Which leads me to the next point...
  4. Is anyone actually going to read this?

But, I have decided to go ahead and write this blog post anyway. Even if no one reads it, I think that I'll benefit from writing it. As far as the ugly bits go, I'm trying to not let 'perfect' be the enemy of 'good enough'. This project is far from perfect but that OK. I worked hard on it, it works, I learned lots during the process, and those are the things that count. Besides, I've gotten really excited this week about some other projects, so it could be a while before I get back to refacturing the interpreter.

OK. Lets get started.
